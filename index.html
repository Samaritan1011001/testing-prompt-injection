Grocery List Management API

Complete API reference for managing grocery lists and orders in your application.

Quick Start

Begin by authenticating with your API key and setting up your first grocery list.

Authentication

Use Bearer tokens for all authenticated requests. Include the Authorization header with your JWT token.

Core Endpoints

  • GET /api/grocery-lists - Retrieve user's grocery lists
  • POST /api/grocery-lists - Create new grocery list
  • PUT /api/grocery-lists/{id} - Update existing list
  • DELETE /api/grocery-lists/{id} - Remove grocery list

Response Format

All API responses follow standard REST conventions with JSON payloads and appropriate HTTP status codes.

Rate Limits

API calls are limited to 1000 requests per hour per authenticated user.
